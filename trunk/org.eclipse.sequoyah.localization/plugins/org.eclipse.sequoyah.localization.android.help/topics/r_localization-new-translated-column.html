New Translated Column dialog

Allows you to translate a complete set of text strings from one language to another.

This dialog appears when you right-click a text string in the Android Localization Files Editor and select Translate column from the menu that appears. It allows you to specify how all of the text strings in the selected column are to be translated to a new language.

Localization file ID The name of the localization strings file to be created for the translated text strings. Note that this file must not already exist. When Automatically add language ID to column name is selected, this field is automatically filled in based upon the target language selected.
Translator The translation service to be used to translate the selected set of text strings to the new language.
Translate from The language the selected strings are in; the translation service translates from the language indicated here to the target language.
Translate into The target language: the language into which the selected strings are to be translated.
Automatically add language ID to column name If selected, the language ID corresponding to the selected target language is appended to the target localization file name.
